## 🧭 G서식 변신 미리보기 (범정부오피스 + Gemini + GPT-5 협업 버전)

> “언제까지 한글(HWP) 안에서 긴~~~ 계획서를 위아래로 왔다리 갔다리 하면서 편집까지 신경쓰실 건가요?”

**이 앱은 편집에 아예 신경 꺼버리고, 오직 글쓰기의 흐름과 논리에만 집중할 수 있게 도와주는 “미리보기용 에디터”입니다.**
범정부오피스(G-서식 변신) 규칙에 맞춘 텍스트를 붙여 넣기만 하면, 즉시 오른쪽에서 완벽한 보고서 스타일로 렌더링됩니다.

---

### 💡 핵심 철학

* **Writing First. Formatting Later.**
  문서의 형식이 아니라 **내용의 논리 흐름(flow)** 에 집중하도록 설계되었습니다.
  (편집은 나중에, 사고는 지금 바로.)
* **한 줄 입력, 멀티라인 표시.**
  G서식 규칙을 깨지 않고도 화면에서는 자동 줄바꿈으로 보기 편하게 표시됩니다.
* **AI-Friendly Workflow.**
  GPT-5나 Gemini에서 생성한 ‘G서식 프롬프트 결과 텍스트’를 그대로 붙여넣기만 하면 완벽히 미리보기 가능.

---

### ⚙️ 주요 기능

| 기능                       | 설명                                                         |
| ------------------------ | ---------------------------------------------------------- |
| 🧱 **실시간 미리보기**          | 왼쪽 입력창의 원본 텍스트가 오른쪽에서 즉시 HWP 보고서 스타일로 렌더링됩니다.              |
| 🧩 **토큰 자동 정리**          | `제목 :` → `제목:` , `원 :` → `원:` 등 전각·공백 자동 수정.               |
| 🌈 **HWP 스타일 모사**        | `제목:`은 하늘색 박스, `배경:`은 흰색 박스, `소제목:`은 네이비 라벨 등 실제 한글 서식 느낌. |
| 📑 **장·소제목 접기/펼치기**      | 긴 보고서를 장(Ⅰ, Ⅱ, Ⅲ…) 또는 소제목 단위로 접었다 펼칠 수 있음.                 |
| ⌨️ **소프트 줄바꿈**           | 한 줄짜리 G서식 텍스트를 자동 줄바꿈으로 표시(가로 스크롤 없음).                     |
| 🌗 **라이트/다크 모드**         | 어두운 환경에서도 눈부심 없이 사용 가능.                                    |
| 📊 **통계/경고 표시**          | 장·소제목·블록 개수 카운트, 누락 토큰 자동 경고.                              |
| 🖨️ **인쇄/PDF 내보내기**      | 미리보기 화면 그대로 출력 또는 PDF 저장.                                  |
| 🧭 **Shift+Click 단일 보기** | 장 제목을 Shift+클릭 시 해당 장만 열리고 나머지는 모두 접힘.                     |

---

### 🪄 사용 방법

1. 왼쪽에 **G서식 프롬프트 결과 텍스트**를 붙여넣습니다.
   (예: `제목:/배경:/소제목:/네모:/원:/바:/당구장:/별:` 형태)
2. **[토큰 정리]** 버튼을 한 번 눌러 공백 및 전각콜론을 정리합니다.
3. 오른쪽에서 자동 미리보기를 확인하면서 **글의 논리 흐름만 수정**합니다.
4. 완성되면 **[입력 복사] → 한글(HWP)에 붙여넣기 → 전체선택 → G서식 변신 클릭!**

### 🧠 철저히 “생각 중심” 도구

> 이 앱은 “보고서 서식”을 미리 맞추는 툴이 아닙니다.
> “보고서의 생각 구조”를 시각적으로 확인하면서 **편집 스트레스 없이 사고 흐름만 점검하는 도구**입니다.

* 🔍 **논리 검증용**: 장별/소제목별 구문 흐름이 자연스러운지 바로 눈으로 확인 가능
* 📘 **팀 협업용**: AI가 생성한 초안을 팀원들과 ‘글 구조’ 중심으로 검토할 때 활용
* 🧩 **AI 통합 워크플로우**: Gemini/NotebookLM/GPT-5/Perplexity 등과 연결해 RAG형 보고서 자동화 가능

* HTML 한 파일만 로컬에 저장 → 더블클릭하면 바로 실행 (인터넷 불필요)
* 브라우저는 **Chrome / Edge / Safari 최신버전** 권장
* 모든 데이터는 **로컬 저장(localStorage)** → 개인정보 및 원고 외부 전송 없음
